# Simulación de Campo Eléctrico en una Placa con Cargas Puntuales

Este proyecto simula el campo eléctrico generado por cargas puntuales en una placa metálica rectangular representada por una matriz 2D. La simulación calcula y muestra el campo eléctrico en cada celda de la placa en función de la distancia de cada carga.

## Requisitos

- CMake
- g++ (GCC)
- Un entorno Linux para la compilación (o Windows con WSL)
